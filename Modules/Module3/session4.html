1. Ler conteúdo de um ficheiro

Para carregar/ler um ficheiro de texto, usamos o comando Get-Content.

$linhas = Get-Content "C:\exemplo\ficheiro.txt"

Este comando lê o conteúdo linha a linha e devolve um array de strings, onde cada linha do ficheiro é um elemento do array.

$linhas.GetType().Name
# Resultado: Object[] (um array de strings)

Como o conteúdo está em array, podemos aplicar operações a cada linha individualmente. Por exemplo, mostrar todas as linhas em maiúsculas:

foreach ($linha in $linhas) {
    $linha.ToUpper()
}

Também podemos contar o número total de linhas:

$total = $linhas.Count
Write-Output "Número de linhas: $total"

2. Procurar texto dentro de ficheiros

Para procurar texto dentro de um ou mais ficheiros usamos Select-String. Este comando funciona como o grep em Linux.

Select-String -Path "C:\exemplo\ficheiro.txt" -Pattern "erro"

O resultado inclui:

  • A linha onde foi encontrado
  • O número da linha
  • O texto correspondente

Podes guardar os resultados numa variável e analisar:

$resultados = Select-String -Path "ficheiro.txt" -Pattern "erro"
$resultados.Count
# Número de ocorrências encontradas

$resultados[0].Line
# Conteúdo da primeira linha com "erro"

3. Procurar por texto com expressões regulares

O comando Select-String aceita não só palavras fixas, mas também expressões regulares no parâmetro -Pattern.

Select-String -Path "ficheiro.txt" -Pattern "^ERRO"

Neste exemplo, o ^ indica que a palavra “ERRO” deve estar no início da linha.

Por omissão, o padrão é tratado como expressão regular.
Ou seja, . representa “qualquer caractere”, \d representa “dígito”, etc.

Se quiseres procurar por texto literal com caracteres especiais, escapa-os com \ ou usa -SimpleMatch:

Select-String -Path "ficheiro.txt" -Pattern "[ERRO]" -SimpleMatch

Também podes aplicar filtros por nome de ficheiro diretamente:

Select-String -Path "C:\logs\*.log" -Pattern "timeout"

Isto aplica o padrão a todos os ficheiros .log da pasta.

4. Iterar sobre vários ficheiros

Se quiseres aplicar os mesmos comandos (como Get-Content ou Select-String) a vários ficheiros, podes usar Get-ChildItem para obter a lista e depois um ciclo foreach para processar cada um.

$ficheiros = Get-ChildItem -Path "C:\logs" -Filter *.txt

foreach ($f in $ficheiros) {
    $linhas = Get-Content $f.FullName
    Write-Output "$($f.Name): $($linhas.Count) linhas"
}

4.1. Renomear ficheiros automaticamente

O comando Rename-Item permite alterar nomes de ficheiros. Podemos usá-lo em conjunto com Get-ChildItem para renomear ficheiros em lote.

$ficheiros = Get-ChildItem -Path "C:\relatorios" -Filter *.txt

foreach ($f in $ficheiros) {
    $novoNome = "2024_" + $f.Name
    Rename-Item -Path $f.FullName -NewName $novoNome
}

⚠️ Atenção: Rename-Item não impede que sobrescrevas ficheiros com o mesmo nome.
Para testar o que o comando faria sem efetuar alterações reais, podes usar o parâmetro -WhatIf:

Rename-Item -Path $f.FullName -NewName $novoNome -WhatIf

O -WhatIf mostra no terminal o que o comando faria, sem realmente executar a ação e pode ser usado com vários outros comandos como Remove-Item, Move-Item, Copy-Item ou Stop-Service.

4.2. Organizar ficheiros por extensão

Podemos automatizar a criação de subpastas por extensão e mover ficheiros para lá.

$pasta = "C:\downloads"
$ficheiros = Get-ChildItem -Path $pasta -File

foreach ($f in $ficheiros) {
    $ext = $f.Extension.TrimStart(".")
    $destino = Join-Path $pasta $ext

    if (-not (Test-Path $destino)) {
        New-Item -ItemType Directory -Path $destino | Out-Null
    }

    Move-Item -Path $f.FullName -Destination $destino
}

Exemplo: imagem.jpg vai para C:\downloads\jpg\

Notas:

  • TrimStart("x"): remove todos os caracteres "x" do início da string.
    Exemplo:

      "///ficheiro.txt".TrimStart("/")    # Resultado: "ficheiro.txt"

  • TrimEnd("x"): remove todos os caracteres "x" do fim da string.
    Exemplo:

      "nome.exe...".TrimEnd(".")          # Resultado: "nome.exe"

  • Trim(): sem argumentos, remove todos os espaços em branco do início e do fim da string.
    O mesmo comportamento acontece com TrimStart() e TrimEnd() se forem chamados sem argumentos:

      "  exemplo  ".Trim()         # Resultado: "exemplo"
  "  exemplo  ".TrimStart()    # Resultado: "exemplo  "
  "  exemplo  ".TrimEnd()      # Resultado: "  exemplo"

    Estes métodos não afetam o conteúdo no meio da string apenas o início e o fim. São úteis para limpar dados recebidos de ficheiros, inputs ou linhas em branco.

  • New-Item permite criar ficheiros ou pastas.
    Usa-se o parâmetro -ItemType para definir o que se quer criar:

      New-Item -Path "C:\exemplo\ficheiro.txt" -ItemType File
  # Cria um ficheiro chamado "ficheiro.txt" no caminho indicado

  New-Item -Path "C:\exemplo\NovaPasta" -ItemType Directory
  # Cria uma nova pasta chamada "NovaPasta"

    O nome do ficheiro ou da pasta é incluído diretamente no valor do parâmetro -Path.

  • Move-Item: move o ficheiro para a nova pasta.

5. 🧩 Exercícios

5.1. 🧩 Exercício – Contar linhas num ficheiro

  1. Pede ao utilizador o caminho de um ficheiro (Read-Host).
  2. Usa Get-Content para ler o ficheiro.
  3. Mostra o número total de linhas.
Resposta 
$ficheiro = Read-Host "Introduz o caminho do ficheiro"
$linhas = Get-Content $ficheiro
Write-Output "Número de linhas: $($linhas.Count)"

5.2. 🧩 Exercício – Procurar palavra num ficheiro

  1. Pede ao utilizador o caminho de um ficheiro.
  2. Pede uma palavra a procurar.
  3. Usa Select-String para encontrar essa palavra.
  4. Mostra o número total de ocorrências.
Resposta 
$ficheiro = Read-Host "Caminho do ficheiro"
$palavra = Read-Host "Palavra a procurar"

$ocorrencias = (Select-String -Path $ficheiro -Pattern $palavra).Count
Write-Output "Ocorrências encontradas: $ocorrencias"

5.3. 🧩 Exercício – Renomear ficheiros com prefixo

  1. Pede ao utilizador uma pasta.
  2. Renomeia todos os ficheiros .log da pasta adicionando o prefixo backup_.
Resposta 
$pasta = Read-Host "Caminho da pasta"
$ficheiros = Get-ChildItem -Path $pasta -Filter *.log

foreach ($f in $ficheiros) {
    $novoNome = "backup_" + $f.Name
    Rename-Item -Path $f.FullName -NewName $novoNome
}

5.4. 🧩 Exercício – Criar relatório em CSV

  1. Pede ao utilizador o caminho de uma pasta.
  2. Gera um CSV com o nome, tamanho e data de modificação de todos os ficheiros.
  3. Guarda o ficheiro como relatorio.csv dentro da mesma pasta.
Resposta 
$pasta = Read-Host "Caminho da pasta"
$ficheiros = Get-ChildItem -Path $pasta -File -Recurse

$relatorio = $ficheiros | Select-Object Name, Length, LastWriteTime

$caminhoCSV = Join-Path $pasta "relatorio.csv"
$relatorio | Export-Csv -Path $caminhoCSV -NoTypeInformation

Write-Output "Relatório criado em: $caminhoCSV"

6. Substituir conteúdo em ficheiros com -replace

O operador -replace permite substituir texto dentro de strings.
Pode ser usado com Get-Content e Set-Content para alterar o conteúdo de um ficheiro.

6.1. Exemplo: Substituir todas as ocorrências de “foo” por “bar” 

(Get-Content "arquivo.txt") -replace 'foo', 'bar' | Set-Content "arquivo.txt"

Este comando faz:

  1. Lê o conteúdo do ficheiro (Get-Content)
  2. Substitui todas as ocorrências da palavra “foo” por “bar” com -replace
  3. Grava novamente o resultado no mesmo ficheiro (Set-Content)

⚠️ Isto sobrescreve o ficheiro original. Usa com cuidado!

6.2. Exemplo com ficheiro de backup: 

(Get-Content "original.txt") -replace "abc", "123" | Set-Content "modificado.txt"

6.3. 🧩 Exercício – Corrigir palavras

Num ficheiro chamado relatorio.txt, substitui todas as ocorrências de:

  • “erado” por “errado”
  • “funcioario” por “funcionário”

7. ✅ Resumo e Boas Práticas

7.1. 📌 Comandos principais abordados

Comando Utilidade principal
Get-Content Ler ficheiros de texto linha a linha
Select-String Procurar palavras ou padrões dentro de ficheiros
Get-ChildItem Obter listas de ficheiros e pastas
Rename-Item Alterar nomes de ficheiros ou pastas
Move-Item Mover ficheiros entre pastas
New-Item Criar ficheiros ou pastas
Export-Csv Gravar objetos num ficheiro CSV
Trim() Remover espaços ou caracteres indesejados

7.2. ✅ Boas práticas

  • Usa -WhatIf sempre que estiveres a alterar, mover ou apagar ficheiros, para testares o que o comando vai fazer.
  • Usa Get-ChildItem -File para garantir que estás a trabalhar apenas com ficheiros (e não pastas).
  • Para evitar erros, valida sempre se a pasta ou ficheiro existe com Test-Path.
  • Quando estiveres a trabalhar com muitos ficheiros, usa -Recurse com cuidado — especialmente se usares Remove-Item.
  • Lembra-te que Select-String usa expressões regulares por padrão — se não quiseres esse comportamento, usa -SimpleMatch.
