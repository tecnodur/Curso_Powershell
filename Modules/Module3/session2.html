🧭 Sessão 2: Manipulação e Transformação de Coleções


🎯 Objetivo Nesta sessão, vais aprender a trabalhar com coleções de objetos no PowerShell. Vamos explorar comandos usados para filtrar, selecionar, transformar, ordenar e agrupar dados — tudo através do pipeline.

Como funcionam estes comandos no geral?

Quando executas comandos como Get-Process, Get-Service, ou Get-ChildItem, o resultado é uma coleção de objetos — ou seja, uma lista com vários elementos.

No PowerShell, podemos passar essa coleção para o próximo comando usando o símbolo | (pipe).

Lado esquerdo → gera objetos
Lado direito → transforma, filtra ou apresenta esses objetos

Exemplo:

Get-Service | Where-Object { $_.Status -eq "Running" }

Aqui o Get-Service gera uma lista com todos os serviços.
O Where-Object filtra cada serviço individualmente, mantendo apenas os que estão ativos.

Comportamento do pipeline

O pipe (|) envia um objeto de cada vez para o comando seguinte.
Esse objeto é representado pela variável automática $_.

Isto permite:

  • Filtrar com Where-Object
  • Selecionar propriedades com Select-Object
  • Aplicar uma ação com ForEach-Object
  • Ordenar com Sort-Object
  • Agrupar com Group-Object

Exemplo:

Get-Process 
| Where-Object { $_.CPU -gt 5 } 
| Select-Object Name, CPU 
| Sort-Object CPU -Descending
  1. Get-Process → Lista todos os processos
  2. Where-Object → Mantém só os que têm mais de 5 segundos de CPU
  3. Select-Object → Escolhe só os campos que interessam
  4. Sort-Object → Ordena por uso de CPU

Cada passo transforma a coleção anterior.

🛠️ Tabela de Comandos de Transformação e Manipulação no Pipeline

Tipo de Uso Comando Descrição Exemplo
Filtro Where-Object Filtra objetos com base numa condição. Get-Service \| Where-Object { $_.Status -eq "Running" }
Seleção Select-Object Seleciona propriedades específicas e/ou limita o número de objetos. Get-Process \| Select-Object Name, CPU -First 5
Ação ForEach-Object Aplica uma ação a cada item. 1..5 \| ForEach-Object { $_ * 2 }
Estatísticas Measure-Object Conta, soma, calcula média, mínimo, máximo. Get-ChildItem \| Measure-Object Length -Sum
Agrupamento Group-Object Agrupa objetos por valor de uma propriedade. Get-Service \| Group-Object Status
Ordenação Sort-Object Ordena objetos por uma ou mais propriedades. Get-Process \| Sort-Object CPU -Descending
Exportação Export-Csv Exporta objetos para ficheiro CSV. Get-Process \| Select-Object Name, Id \| Export-Csv proc.csv
Output Texto Out-File Escreve output para ficheiro de texto. "Mensagem" \| Out-File log.txt
Formatação Format-Table Apresenta o output como tabela formatada. Get-Service \| Format-Table Name, Status
Formatação Format-List Apresenta o output como lista detalhada. Get-Process \| Format-List *
Duplicação Tee-Object Envia o output para o próximo comando e também para ficheiro/variável. Get-Process \| Tee-Object -FilePath proc.txt \| Sort-Object CPU
Junção Texto Join-String Junta strings com separador (PowerShell 7+). Get-Service \| Join-String -Property Name -Separator ", "
Comparação Compare-Object Compara duas coleções e mostra diferenças. Compare-Object $lista1 $lista2
Exploração Get-Member Mostra propriedades e métodos dos objetos recebidos. Get-Process \| Get-Member

Alguns exemplos práticos

Where-Object

Filtra apenas os processos com mais de 100 MB de memória:

Get-Process | Where-Object { $_.WorkingSet -gt 100MB }

Select-Object

Seleciona propriedades específicas:

Get-Process | Select-Object Name, CPU, Id

ForEach-Object

Mostra o nome em maiúsculas:

"nuno","ana","rui" | ForEach-Object { $_.ToUpper() }

Sort-Object

Ordena os serviços por estado:

Get-Service | Sort-Object Status

Group-Object

Agrupa serviços por estado:

Get-Service | Group-Object Status

✅ Boas Práticas

  • Usa Select-Object para limitar o output e melhorar a legibilidade
  • Usa Where-Object com expressões claras e legíveis
  • Evita ForEach-Object quando podes resolver com expressões simples

🧠 Exercícios

1. Mostrar apenas os serviços que estão a correr

Queremos listar apenas os serviços que têm o estado “Running”.

2. Obter os 5 processos com mais CPU

Precisamos de encontrar os processos que mais têm usado CPU no sistema.

3. Converter uma lista de nomes para maiúsculas com ForEach-Object

Dada uma lista de nomes, queremos mostrar cada um deles em letras maiúsculas.

4. Agrupar processos pelo nome e contar quantos existem de cada

Queremos saber quantas instâncias existem de cada processo com base no nome.

🧭 Recomendações Finais

  • Esta sessão é base para criação de relatórios e scripts mais avançados.
  • A combinação Where + Select + Export é um padrão muito comum.
Get-Process | Where-Object { $_.CPU -gt 10 } | 
Select-Object Name, CPU | 
Export-Csv "processos.csv" -NoTypeInformation
