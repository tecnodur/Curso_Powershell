1. Introdução aos Loops

Em programação, muitas vezes precisamos de repetir ações: processar vários itens, tentar até uma condição ser verdadeira, validar inputs, etc.

Em vez de copiar e colar código várias vezes, usamos loops.

“Enquanto (ou até que) uma condição for verdadeira, repete este bloco de código.”

2. O loop for

O for é usado quando sabemos de antemão quantas vezes queremos repetir.

Sintaxe: 

for (<inicialização>; <condição>; <incremento>) {
    # Código a repetir
}
  • Inicialização: define a variável de controlo (ex: $i = 0).
  • Condição: quando for falsa, o loop termina.
  • Incremento: atualiza a variável em cada repetição.

Exemplo: contar de 1 a 5 

for ($i = 1; $i -le 5; $i++) {
    Write-Output "Valor atual: $i"
}

3. O loop foreach

O foreach é usado para percorrer todos os itens de uma coleção (array, lista, etc.).

Sintaxe: 

foreach ($item in $colecao) {
    # Código a executar para cada item
}

Exemplo: listar nomes 

$nomes = @("Ana", "Bruno", "Carlos")

foreach ($nome in $nomes) {
    Write-Output "Nome: $nome"
}

Alternativa: usar 1..10 para gerar uma lista de números

Se quiseres simplesmente criar um conjunto de números sequenciais (por exemplo, de 1 a 10), podes usar o operador .. (range):

$numeros = 1..10
foreach ($n in $numeros) {
    Write-Output "Número: $n"
}

ℹ1..10 cria automaticamente um array com os números de 1 a 10.
É uma forma rápida e prática de gerar sequências numéricas. —

4. O loop while

O while executa enquanto uma condição for verdadeira.

Sintaxe: 

while (<condição>) {
    # Código a repetir
}
  • Se a condição for falsa desde o início, o while não executa nenhuma vez.

Exemplo: contar até 3 

$i = 1
while ($i -le 3) {
    Write-Output "Contagem: $i"
    $i++
}

5. O loop do-until

O do-until executa sempre pelo menos uma vez e repete até a condição ser verdadeira.

Sintaxe: 

do {
    # Código a repetir
} until (<condição>)
  • Inverso do while: o loop corre até a condição ser verdadeira.

Exemplo: pedir número até ser maior que 10 

do {
    $numero = Read-Host "Introduz um número maior que 10"
} until ([int]$numero -gt 10)

Write-Output "Obrigado! Introduziste: $numero"

6. Comandos especiais em loops: break e continue

Dentro de loops, podemos usar dois comandos especiais para controlar o fluxo de execução:

break

  • Interrompe o loop imediatamente.
  • O código salta para depois do loop.

Exemplo: parar ao encontrar um valor

$nomes = @("Ana", "Bruno", "Carlos")

foreach ($nome in $nomes) {
    if ($nome -eq "Bruno") {
        break
    }
    Write-Output "Nome: $nome"
}

Neste exemplo:

  • O loop para imediatamente quando encontra “Bruno”.
  • “Carlos” nunca é mostrado.

continue

  • Salta para a próxima iteração do loop, sem terminar o loop.
  • O código depois do continue dentro do loop não é executado.

Exemplo: ignorar um valor

$nomes = @("Ana", "Bruno", "Carlos")

foreach ($nome in $nomes) {
    if ($nome -eq "Bruno") {
        continue
    }
    Write-Output "Nome: $nome"
}

Neste exemplo:

  • “Bruno” é ignorado.
  • “Ana” e “Carlos” são mostrados.

🧠 Dicas práticas

Comando Uso principal
break Parar o loop totalmente
continue Saltar apenas a iteração atual

7. 🧩 Exercícios

🧩 Exercício – Script contagem.ps1 com for

Cria um script contagem.ps1 que:

  • Conta de 10 até 1 (a descer) usando for.
  • Mostra cada número com Write-Output.
Resposta 
# contagem.ps1
# Conta de 10 até 1

for ($i = 10; $i -ge 1; $i--) {
    Write-Output "Número: $i"
}

🧩 Exercício – Script listar_processos.ps1 com foreach

Cria um script listar_processos.ps1 que:

  • Usa Get-Process para obter a lista de processos.
  • Mostra o nome de cada processo usando foreach.
Resposta 
# listar_processos.ps1
# Lista todos os processos ativos

$processos = Get-Process

foreach ($proc in $processos) {
    Write-Output "Processo: $($proc.Name)"
}

🧩 Exercício – Script servicos_parados.ps1 com foreach

Cria um script servicos_parados.ps1 que:

  • Usa Get-Service.
  • Mostra apenas os serviços que estão parados (Status -eq "Stopped").
Resposta 
# servicos_parados.ps1
# Lista serviços parados no sistema

$servicos = Get-Service

foreach ($svc in $servicos) {
    if ($svc.Status -eq "Stopped") {
        Write-Output "Serviço parado: $($svc.Name)"
    }
}

🧩 Exercício – Script esperar_minuto.ps1 com while

Cria um script esperar_minuto.ps1 que:

  • Usa while para esperar até o minuto atual ser múltiplo de 5.
  • Mostra o minuto quando for encontrado.
Resposta 
# esperar_minuto.ps1
# Espera até o minuto atual ser múltiplo de 5

$minuto = (Get-Date).Minute

while ($minuto % 5 -ne 0) {
    Write-Output "Minuto atual: $minuto"
    Start-Sleep -Seconds 10
    $minuto = (Get-Date).Minute
}

Write-Output "Minuto múltiplo de 5 encontrado: $minuto"

🧩 Exercício – Script encontrar_notepad.ps1 com do-until

Cria um script encontrar_notepad.ps1 que:
De 5 em 5 segundos, verifica se o processo “notepad” está aberto.
Enquanto o Notepad não estiver aberto, o script deve:

  • Escrever no terminal que o Notepad ainda não foi encontrado.

Assim que o Notepad estiver aberto, o script deve:

  • Escrever no terminal que o Notepad foi encontrado.
  • Terminar a execução.
Resposta 
# encontrar_notepad.ps1
# Procura o processo notepad até o encontrar

do {
    $proc = Get-Process -Name "notepad" -ErrorAction SilentlyContinue
    if (-not $proc) {
        Write-Output "Notepad ainda não encontrado..."
        Start-Sleep -Seconds 5
    }
} until ($proc)

Write-Output "Processo notepad encontrado!"

🧩 Exercício – Script ignorar_pares.ps1 com continue

Cria um script ignorar_pares.ps1 que:

  • Cria um array com os números de 1 a 10.
  • Usa foreach para mostrar apenas os números ímpares.
Resposta 
# ignorar_pares.ps1
# Mostra apenas números ímpares de 1 a 10

$numeros = 1..10

foreach ($n in $numeros) {
    if ($n % 2 -eq 0) {
        continue
    }
    Write-Output "Número ímpar: $n"
}

8. Filtrar listas com Where-Object

Quando tens uma lista de elementos (ex: nomes, números, processos), podes usar o comando Where-Object para filtrar apenas os elementos que interessam.

📌 Sintaxe básica 

$lista | Where-Object { condição }

Dentro das chavetas {}, usamos o símbolo especial $_ que representa o elemento atual da lista.

🔍 Exemplo 1 – Números maiores que 10 

$numeros = 5, 8, 12, 20

$maiores = $numeros | Where-Object { $_ -gt 10 }

Write-Output $maiores
  • $_ representa cada número da lista.
  • A condição $_ -gt 10 devolve apenas os que são maiores que 10.

🔍 Exemplo 2 – Palavras que começam por “A” 

$palavras = "Ana", "Bruno", "alface", "Árvore"

$comA = $palavras | Where-Object { $_ -cmatch '^a' }

Write-Output $comA
  • O -cmatch faz a correspondência case-sensitive.

Dica: $_.Propriedade em objetos

Se estiveres a trabalhar com objetos (ex: resultado de Get-Process), podes usar $_ seguido da propriedade:

Get-Process | Where-Object { $_.CPU -gt 50 }

🧩 Exercício – Filtrar frutas cítricas

  1. Cria uma lista de frutas: laranja, banana, limão, kiwi.
  2. Usa Where-Object para obter apenas as frutas que terminam em ão.
Resposta 
$frutas = "laranja", "banana", "limão", "kiwi"

$citricas = $frutas | Where-Object { $_ -like "*ão" }

Write-Output $citricas

🧩 Exercício – Números pares

  1. Cria uma lista com os números de 1 a 10.
  2. Usa Where-Object para filtrar apenas os números pares (divisíveis por 2).
Resposta 
$numeros = 1..10

$pares = $numeros | Where-Object { $_ % 2 -eq 0 }

Write-Output $pares

🧩 Exercício – Processos com muita memória

  1. Usa Get-Process para obter os processos.
  2. Filtra os que usam mais de 200 MB de memória (WorkingSet > 200MB).
    • Dica: 200 MB = 200 * 1024 * 1024
Resposta 
Get-Process | Where-Object { $_.WorkingSet -gt (200 * 1024 * 1024) }

9. Resumo e Boas Práticas

✅ Usa for quando souberes quantas vezes vais repetir.
✅ Usa foreach para percorrer coleções.
✅ Usa while quando precisas de verificar antes de correr.
✅ Usa do-until quando queres executar pelo menos uma vez.
