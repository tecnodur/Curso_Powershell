1. Trabalhar com Strings

Em PowerShell, strings são simplesmente sequências de caracteres (texto) que podemos manipular facilmente.

1.1. Concatenar Strings

Concatenar significa juntar duas ou mais strings numa só.

Métodos comuns:

  • Interpolação (recomendado):

      $nome = "Nuno"  
  $mensagem = "Olá, $nome!"  
  Write-Output $mensagem

  • Concatenação usando +:

      $nome = "Nuno"
  $mensagem = "Olá, " + $nome
  Write-Output $mensagem

Nota: A interpolação (usar $variavel dentro de "texto") é mais simples e legível.

1.2. Alterar Maiúsculas e Minúsculas

Podes facilmente converter uma string para maiúsculas ou minúsculas.

  • Para maiúsculas (ToUpper()):

      $texto = "PowerShell"
  $texto.ToUpper()  # Resulta em "POWERSHELL"

  • Para minúsculas (ToLower()):

      $texto = "PowerShell"
  $texto.ToLower()  # Resulta em "powershell"

Estes métodos não alteram a variável original, apenas devolvem a nova string.
Se quiseres guardar o resultado, tens de atribuir a uma variável:

$texto = $texto.ToUpper()

1.3. Dividir uma String (-split)

Usamos o operador -split para dividir uma string em várias partes.

  • Exemplo básico:

      $frase = "Maçã;Banana;Cereja"
  $frutas = $frase -split ";"
  $frutas

Resultado:

Maçã
Banana
Cereja

O operador -split gera um array com as partes da string.
$frutas[1] será Banana

1.4. Substituir Texto (-replace)

Para substituir texto numa string, usamos -replace.

  • Sintaxe:

      $novaString = $textoAntigo -replace "texto a procurar", "novo texto"
  • O operador -replace aplica-se à string original e gera uma nova string como resultado.

  • O texto a procurar é interpretado como uma expressão regular simples.

  • Exemplo:

      $frase = "Olá Mundo!"
  $novaFrase = $frase -replace "Mundo", "PowerShell"
  Write-Output $novaFrase

    Resultado:

          Olá PowerShell!

Exemplo com Expressão Regular

Também podes usar -replace para fazer substituições com base em padrões, não apenas texto fixo.

Exemplo: Substituir todos os números por #

$texto = "Encomenda 123 enviada no dia 24."
$novoTexto = $texto -replace '\d+', '#'
Write-Output $novoTexto

Resultado:

Encomenda # enviada no dia #.

Explicação:

  • \d+ significa “um ou mais dígitos”.
  • O -replace procura todos os números (123, 24) e substitui cada ocorrência por #.
  • Como o -replace interpreta o primeiro argumento como expressão regular, padrões como \d+ funcionam diretamente.

Usa aspas simples '...' à volta do padrão para evitar que o PowerShell tente expandir \d

1.5. Strings Multilinha e Caracteres Especiais

Multilinha

Para criar strings em várias linhas, usa aspas duplas normais:

    $texto = @"
Este é um texto
em várias linhas
no PowerShell.
"@
  • O @ é necessário para indicar ao PowerShell que se trata de uma here-string, não de uma string normal.
  • Começa com @" e fecha com "@
  • Tudo entre estes delimitadores é considerado uma única string.

Caracteres especiais

Dentro de strings podes usar alguns símbolos especiais:

Símbolo Significado
`n Nova linha
`t Tabulação (tab)
`r Retorno ao início da linha

Exemplo:

$mensagem = "Linha 1`nLinha 2`nLinha 3"
Write-Output $mensagem

Resultado:

Linha 1  
Linha 2  
Linha 3

O acento grave (`) é usado para introduzir caracteres especiais.

2. Trabalhar com Números

Em PowerShell, já vimos como fazer operações aritméticas básicas (+, -, *, /, %) e como usar operadores de comparação.

Se quiseres rever, consulta:
🔗 Session 2 — Operadores Básicos

Nesta secção, vamos focar-nos apenas em como converter entre números e texto, algo muito útil em scripts.

2.1. Converter Número para Texto e Texto para Número

Em PowerShell, podes converter facilmente entre números e texto.

Converter Número para Texto

Para incluir números em texto, podes:

  • Usar interpolação:

      $idade = 25
  $mensagem = "Tenho $idade anos."
  Write-Output $mensagem

  • Usar .ToString() para transformar explicitamente:

      $numero = 123
  $texto = $numero.ToString()
  Write-Output "Texto: $texto"

.ToString() converte o número para uma representação textual.

Converter Texto para Número

Se receberes um valor como texto (ex: de um input), podes convertê-lo para número usando [int], [double] ou [decimal]:

$inputTexto = "42"
$numero = [int]$inputTexto
Write-Output ($numero + 10)   # Soma 10 ao número

⚠️ Se o texto não for um número válido, a conversão gera erro.

  3. Multiplica o número por 2.
  4. Mostra a mensagem: “O dobro de é ."

💡 Dica: Usa [int] para a conversão e interpolação de strings para construir a frase.

3. Trabalhar com Arrays Simples

Já aprendemos como criar e aceder a arrays simples (@()) e como trabalhar com o seu conteúdo.
Se quiseres rever, consulta:
🔗 Session 2 — Tipos Compostos

Nota: Tudo o que vamos ver neste ponto também funciona com ArrayList (objetos criados com New-Object System.Collections.ArrayList).

Agora vamos focar-nos em algumas operações práticas muito úteis: contar elementos e verificar se um valor existe num array.

3.1. Criar e Aceder a Elementos de um Array

  • Para criar um array:

      $frutas = @("Maçã", "Banana", "Cereja")

  • Para aceder a um elemento (os índices começam em 0):

      $primeira = $frutas[0]
  Write-Output "Primeira fruta: $primeira"

  • Para alterar um elemento:

      $frutas[1] = "Manga"

3.2. Contar Elementos

Para saber quantos elementos tem um array, usa .Count:

$total = $frutas.Count
Write-Output "Total de frutas: $total"

3.3. Verificar a Existência de um Elemento (-contains)

Para verificar se um array contém um determinado valor, usa o operador -contains:

$existe = $frutas -contains "Cereja"
Write-Output "Existe Cereja? $existe"
  • Se o valor existir, o resultado é $true.
  • Se não existir, o resultado é $false.

Atenção: -contains é case-insensitive por omissão.

Como fazer uma verificação Case-Sensitive?

O PowerShell não tem um operador -ccontains.
Se precisares de fazer uma verificação sensível a maiúsculas e minúsculas, deves usar Where-Object com -ceq:

$frutas = @("Maçã", "Banana", "Cereja")
$input = "CEREJA"

$existe = $frutas | Where-Object { $_ -ceq $input }

if ($existe) {
    Write-Output "Encontrado!"
}
else {
    Write-Output "Não encontrado."
}

O -ceq compara respeitando maiúsculas/minúsculas. Se não houver correspondência exata, o resultado é $null.
Nota: Neste exemplo usamos Where-Object, um comando que permite filtrar elementos de uma lista. O símbolo $_ representa o elemento atual da lista. Vamos explorar isto com mais detalhe noutra sessão.

  3. Verifica se a cor existe no array.
  4. Mostra:
    • Se existir: “Cor encontrada!”
    • Se não existir: “Cor não encontrada.”

💡 Dica: Usa -contains para fazer a verificação.

4. Trabalhar com Datas

Já aprendemos que o comando Get-Date devolve a data e hora atuais como um objeto do tipo [datetime].
Se quiseres rever, consulta:
🔗 Session 2 — Tipos Primitivos

Agora vamos explorar formas práticas de trabalhar com datas em PowerShell.

4.1. Obter a Data Atual (Get-Date)

Para obter a data e hora atuais:

$agora = Get-Date
Write-Output $agora

O resultado é um objeto completo que contém:

  • Data
  • Hora
  • Dia da semana
  • Fuso horário
  • Entre outras informações

4.2. Formatar Datas como Texto (.ToString())

Podes converter uma data em texto usando o método .ToString(), especificando o formato que queres.

Exemplos:

  • Formato completo:

      $agora = Get-Date
  $texto = $agora.ToString()
  Write-Output $texto

  • Formatar só a data (dia/mês/ano):

      $dataSimples = (Get-Date).ToString("dd/MM/yyyy")
  Write-Output $dataSimples

  • Formatar data e hora:

      $dataHora = (Get-Date).ToString("dd-MM-yyyy HH:mm")
  Write-Output $dataHora

📋 Principais padrões de formatação

Código Significado Exemplo (com 5 de abril de 2025, 14h30)
dd Dia com dois dígitos 05
d Dia sem zero inicial 5
MM Mês com dois dígitos 04
M Mês sem zero inicial 4
yyyy Ano com quatro dígitos 2025
yy Ano com dois dígitos 25
HH Hora (24h) com dois dígitos 14
hh Hora (12h) com dois dígitos 02
mm Minutos com dois dígitos 30
ss Segundos com dois dígitos 00

Para mais exemplos e códigos disponíveis, consulta:
🔗 Custom Date and Time Format Strings — Documentação Oficial (.NET)

🌍 Usar Cultura (Globalization) para aplicar formatos regionais

O método .ToString() pode receber um segundo argumento que define a cultura (formato regional) da data, incluindo nomes de meses e dias em português.

Para isso, usamos [System.Globalization.CultureInfo].

Exemplo: Formatar a data com nomes em português (Portugal) 

    $pt = [System.Globalization.CultureInfo]::GetCultureInfo("pt-PT")
    $data = Get-Date
    $texto = $data.ToString("dddd, dd 'de' MMMM 'de' yyyy", $pt)
    Write-Output $texto

Resultado esperado:

segunda-feira, 21 de abril de 2025

ℹ️ Isto é útil quando queres apresentar a data com o nome do mês ou do dia da semana em português, independentemente da configuração regional do sistema.

Exemplos de outras culturas possíveis:

Código Localização
pt-PT Português (Portugal)
en-US Inglês (Estados Unidos)
fr-FR Francês (França)
de-DE Alemão (Alemanha)

Dica:

Ao trabalhar com utilizadores de diferentes países, é boa prática indicar explicitamente a cultura quando formatas datas com .ToString() — especialmente se usas dddd ou MMMM.

🔗 Documentação: CultureInfo (Microsoft)

4.3. Adicionar ou Subtrair Dias

O PowerShell permite manipular datas facilmente.

Adicionar dias: 

$amanha = (Get-Date).AddDays(1)
Write-Output "Amanhã: $amanha"

Subtrair dias: 

$ontem = (Get-Date).AddDays(-1)
Write-Output "Ontem: $ontem"
  • .AddDays(1) ➔ adiciona 1 dia.
  • .AddDays(-1) ➔ subtrai 1 dia.

Existem outros métodos como .AddHours(), .AddMinutes(), etc., que funcionam de forma semelhante.

  2. Mostra a data atual no formato “dd-MM-yyyy”.
  3. Mostra a data de aqui a 7 dias também no mesmo formato.

💡 Dica: Usa .AddDays() e .ToString().

5. Conversões de Tipos de Dados

Em PowerShell, podes converter valores de um tipo para outro de forma explícita.
Já aprendemos que é possível usar [tipo]$variavel para forçar o tipo de uma variável.

Agora vamos focar-nos nas conversões mais práticas para o dia a dia.

5.1. Conversões Comuns

Converter String para Número

Se tens um número em formato de texto, podes convertê-lo para [int], [double], ou [decimal].

Exemplo:

$texto = "42"
$numero = [int]$texto
Write-Output ($numero + 10)

Se o texto não representar um número válido, a conversão gera um erro.

Converter Número para String

Para transformar um número em texto:

  • Usar interpolação:

      $numero = 123
  $texto = "$numero"
  Write-Output "Texto: $texto"

  • Ou usar .ToString():

      $numero = 123
  $texto = $numero.ToString()
  Write-Output "Texto: $texto"

Converter String para Data

Se tens uma data como texto, podes convertê-la diretamente para [datetime]:

$texto = "2025-04-21"
$data = [datetime]$texto
Write-Output $data

O formato do texto deve ser compatível com os formatos que o PowerShell consegue interpretar automaticamente.

📋 Formatos mais comuns aceites

Exemplo de Texto Descrição
2025-04-21 Ano-Mês-Dia (ISO 8601)
21/04/2025 Dia/Mês/Ano (regional PT-PT)
04/21/2025 Mês/Dia/Ano (regional US)
April 21, 2025 Nome do mês em inglês
21-Apr-2025 Dia-AbreviaçãoMês-Ano
2025-04-21T14:30:00 ISO 8601 com hora
21-04-2025 14:30 Dia-Mês-Ano Hora:Minuto
Monday, April 21, 2025 Dia da semana, Mês, Dia, Ano

Dica prática:
Para garantir máxima compatibilidade, prefere sempre usar o formato ISO 8601 (yyyy-MM-dd) ao pedir datas em input ou ao fazer conversões no PowerShell.

Exemplo recomendado:
"2025-04-21"

🔗 Documentação oficial: about_DateTime (PowerShell)

🔗 Custom Date and Time Format Strings (.NET)

5.2. Boas Práticas de Conversão

  • Confirma sempre o tipo antes de converter
    Usa .GetType().Name ou operadores -is para verificar se precisas mesmo de converter.

  • Garante que o valor é compatível
    Só tentes converter se souberes que o texto representa um número ou uma data válida.

  • Prefere conversão explícita a conversão automática
    Assim o comportamento do script é mais previsível.

5.3. 🧩 Exercício Prático

Cria um pequeno script que:

  1. Pede ao utilizador para introduzir quantos dias quer adicionar à data atual (Read-Host).
  2. Converte o input para inteiro.
  3. Calcula a nova data.
  4. Mostra a nova data no formato “dddd, dd MMMM yyyy” (exemplo: “sexta-feira, 02 maio 2025”).

💡 Dicas:

  • Usa [int] para converter o input.
  • Usa .AddDays() para calcular a nova data.
  • Usa .ToString() para formatar.

🔗 Documentação oficial: about_Data_Types
