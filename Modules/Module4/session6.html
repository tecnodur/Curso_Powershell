1. O que são bibliotecas e namespaces .NET?

O .NET fornece centenas de funcionalidades agrupadas em bibliotecas, organizadas por namespaces (espaços de nomes).
PowerShell consegue usar muitas dessas funcionalidades diretamente, pois é construído sobre .NET.

Exemplos de namespaces:

  • System.Math
  • System.IO.File
  • System.Net.WebClient
  • System.Windows.Forms

2. Como usar bibliotecas no PowerShell

Muitas funcionalidades já estão carregadas no ambiente PowerShell. Outras precisam de ser carregadas com:

Add-Type -AssemblyName Nome.Da.Biblioteca

Exemplo com interface gráfica:

Add-Type -AssemblyName System.Windows.Forms
[System.Windows.Forms.MessageBox]::Show("Olá do .NET!")

Para saber se uma biblioteca já está carregada:

[AppDomain]::CurrentDomain.GetAssemblies()

3. Usar bibliotecas externas com NuGet

Quando uma biblioteca não está disponível por defeito, podes instalá-la através do NuGet.

  1. Vai a https://www.nuget.org/
  2. Procura a biblioteca (ex: PdfSharp, CsvHelper)
  3. Descarrega o .nupkg
  4. Extrai o .dll da pasta lib/net45 ou semelhante

  5. No PowerShell:

     Add-Type -Path "C:\Bibliotecas\PdfSharp\PdfSharp.dll"

Lê sempre a página do NuGet para confirmar dependências

4. Explorar o que há dentro de uma biblioteca

Podes descobrir métodos e propriedades com:

[System.Math] | Get-Member -Static

Para filtrar:

[Math] | Get-Member -Static | Where-Object Name -like "*Round*"

Para instâncias:

$obj = New-Object System.Text.StringBuilder
$obj | Get-Member

Consulta oficial:
https://learn.microsoft.com/en-us/dotnet/api/

5. O que são métodos estáticos?

Um método estático pertence a uma classe e não precisa de criar objetos para ser usado.

[Classe]::Método()

Exemplos:

[Math]::Sqrt(25)
[DateTime]::Now
[Environment]::UserName

Para descobrir métodos:

[System.Math] | Get-Member -Static

7. Aceder à Internet com .NET

7.1. Usar WebClient

A classe [System.Net.WebClient] permite descarregar páginas, ficheiros e dados de URLs. Muito útil para automatizações e integrações simples com serviços externos.

Exemplo: descarregar conteúdo de uma página 

$web = New-Object System.Net.WebClient
$html = $web.DownloadString("https://example.com")
$html.Substring(0, 300)  # Mostra os primeiros 300 caracteres

Exemplo: guardar uma página num ficheiro 

$web = New-Object System.Net.WebClient
$web.DownloadFile("https://example.com", "C:\Temp\pagina.html")

Exemplo: descarregar imagem ou ficheiro 

$web = New-Object System.Net.WebClient
$url = "https://upload.wikimedia.org/wikipedia/commons/thumb/4/47/PNG_transparency_demonstration_1.png/200px-PNG_transparency_demonstration_1.png"
$destino = Join-Path -Path (Get-Location) -ChildPath "imagem.png"
$web.Headers.Add("User-Agent", "Mozilla/5.0")
$web.DownloadFile($url, $destino)

7.2. Usar HttpClient (moderno)

  • WebClient é simples, mas descontinuado nas versões mais recentes do .NET.
    Para aplicações novas, prefere [System.Net.Http.HttpClient].
  • Mesmo assim, continua útil para scripts rápidos e locais no PowerShell.

Exemplo: descarregar conteúdo de uma página (com HttpClient) 

Add-Type -AssemblyName System.Net.Http
$client = [System.Net.Http.HttpClient]::new()
$html = $client.GetStringAsync("https://example.com").Result
$html.Substring(0, 300)  # Mostra os primeiros 300 caracteres

Exemplo: guardar uma página num ficheiro (com HttpClient) 

Add-Type -AssemblyName System.Net.Http
$client = [System.Net.Http.HttpClient]::new()
$html = $client.GetStringAsync("https://example.com").Result

$caminho = "C:\Temp\pagina.html"
[System.IO.File]::WriteAllText($caminho, $html)

Exemplo: descarregar imagem ou ficheiro (com HttpClient) 

Add-Type -AssemblyName System.Net.Http
$client = [System.Net.Http.HttpClient]::new()
$client.DefaultRequestHeaders.UserAgent.ParseAdd("Mozilla/5.0")

$url = "https://upload.wikimedia.org/wikipedia/commons/thumb/4/47/PNG_transparency_demonstration_1.png/200px-PNG_transparency_demonstration_1.png"
$destino = Join-Path -Path (Get-Location) -ChildPath "imagem.png"

$bytes = $client.GetByteArrayAsync($url).Result
[System.IO.File]::WriteAllBytes($destino, $bytes)

Nota: o .Result força a execução síncrona (bloqueante) para facilitar uso em scripts. Em contextos mais avançados podes usar await com async.

9. Boas práticas

  • Usa [Classe]::Método() sempre que possível
  • Usa Get-Member -Static para descobrir métodos
  • Usa bibliotecas nativas sempre que disponíveis
  • Guarda funções úteis no $PROFILE
  • Consulta a documentação oficial quando precisares de algo específico
