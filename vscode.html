1. O que é o Visual Studio Code?

O Visual Studio Code (VSCode) é um editor de código gratuito, leve e poderoso da Microsoft. É compatível com várias linguagens de programação e tem suporte nativo para PowerShell através de extensões.

Principais vantagens:

  • Grátis e multiplataforma (Windows, Linux, macOS).
  • Extensível com milhares de extensões.
  • Integração com Git, Terminal e IntelliSense.
  • Ideal para escrever, testar e depurar scripts PowerShell.

🔗 Download do VSCode

2. Instalar o VSCode

  1. Acede a https://code.visualstudio.com/
  2. Faz o download da versão para o teu sistema operativo.
  3. Segue os passos da instalação (no Windows, basta clicar em “Seguinte” várias vezes).

💡 Após instalar, poderás abrir o VSCode a partir do menu Iniciar ou escrever code no terminal (se ativares a opção durante a instalação).

3. Instalar a extensão PowerShell

  1. Abre o VSCode.
  2. Vai ao menu lateral esquerdo e clica em Extensões (ícone de quadrado com cantos afastados).
  3. Pesquisa por PowerShell.
  4. Instala a extensão chamada PowerShell (publisher: Microsoft).

🔗 Extensão PowerShell no Marketplace

4. Como usar o VSCode com PowerShell

📁 Abrir uma pasta de scripts

  • Abre uma pasta onde tenhas os teus scripts .ps1.
  • Usa o menu File > Open Folder (Ficheiro > Abrir Pasta).

✍️ Criar um novo ficheiro .ps1

  • Vai a File > New File, guarda como meuscript.ps1.

▶️ Executar scripts

  • Podes executar o script no terminal integrado clicando em Run ou com o atalho F5 (depuração) ou Ctrl+F5 (execução simples).
  • O terminal aparece na parte inferior do ecrã.

🧠 Dicas:

  • Usa Ctrl+ para aumentar e Ctrl- para reduzir o tamanho da fonte.
  • Usa Ctrl+Space para sugestões inteligentes de comandos (IntelliSense).

5. Recomendações de extensões úteis

Extensão Descrição
PowerShell (Microsoft) Suporte oficial à linguagem, linting, execução e depuração.
Path Intellisense Ajuda a completar caminhos de ficheiros e pastas.

6. Configurações úteis no VSCode

Podes configurar algumas opções no ficheiro settings.json ou via interface gráfica.

Sugestões:

{
  "editor.tabSize": 4,                         // Define a tabulação como 4 espaços
  "editor.insertSpaces": true,                 // Usa espaços em vez de tabulações reais
  "files.insertFinalNewline": true,            // Garante uma linha em branco no fim do ficheiro
  "editor.rulers": [80],                       // Mostra uma linha vertical na coluna 80 (boa prática)
  "editor.wordWrap": "on",                     // Ativa quebra automática de linha no editor
  "editor.bracketPairColorization.enabled": true // Ativa cores diferentes para pares de parêntesis/chaves
}

Para aceder às definições JSON:

Prime Ctrl + Shift + P

Escreve Preferences: Open User Settings (JSON)

Cola as opções acima.

7. Atalhos úteis no VSCode

Ação Atalho
Executar script (sem debug) Ctrl + F5
Executar script (com debug) F5
Terminal integrado Ctrl + ‘
Abrir extensões Ctrl + Shift + X
Comando rápido Ctrl + Shift + P
Ver atalhos de teclado Ctrl + K, Ctrl + S

8. Dicas finais

✅ Usa VSCode sempre que possível para escrever os teus scripts: tens validação, sugestões, cores e terminal integrado.
✅ Mantém o terminal PowerShell atualizado: Download PowerShell
✅ Podes também instalar o PowerShell 7+ (mais moderno e multiplataforma).
