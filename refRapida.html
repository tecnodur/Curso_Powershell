Comandos Essenciais em PowerShell

🧠 Ajuda e Exploração

O que fazer Comando(s) Observação
Ver ajuda completa de um comando Get-Help 'comando' -Full ou Help 'comando' -Full Mostra tudo: parâmetros, descrição, exemplos
Ver só os parâmetros de um comando Get-Help 'comando' -Parameter * Lista apenas os parâmetros
Ver métodos e propriedades de um objeto 'objeto' \| Get-Member ou Get-Member -InputObject 'objeto' Mostra métodos e propriedades
Ver propriedades de processos Get-Process \| Get-Member Explora as propriedades disponíveis
Listar todos os comandos disponíveis Get-Command Podes filtrar com -Verb Get, -Noun Service, etc.

📦 Variáveis e Input

O que fazer Comando(s) Observação
Criar ou alterar uma variável $variavel = 'valor' Atribui valores
Ver conteúdo de uma variável $variavel Só digitar o nome no terminal
Ver o tipo de uma variável ($variavel).GetType() Mostra o tipo completo (System.String, etc.)
Ver tipo e valor de uma variável $var.GetType().Name; $var Alternativa rápida
Ver variáveis no ambiente Get-Variable Mostra todas as variáveis atuais
Pedir input ao utilizador Read-Host "Pergunta" Exemplo: $nome = Read-Host "Qual o teu nome?"

⏱️ Tempo e Controlo

O que fazer Comando(s) Observação
Ver tempo atual Get-Date Pode usar .AddDays(), .ToString(), etc.
Pausar o script durante alguns segundos Start-Sleep -Seconds 5 Também funciona com -Milliseconds
Repetir bloco com contador for ($i = 1; $i -le 10; $i++) { ... } Loop clássico com incremento

📊 Manipulação de Coleções (com pipeline)

O que fazer Comando(s) Observação
Filtrar elementos Where-Object { $_ -gt 10 } Usado com | para filtrar coleções
Selecionar propriedades Select-Object Name, Status Ideal após Get-Process, Get-Service, etc.
Aplicar ação a cada item ForEach-Object { $_.ToUpper() } Manipula objetos individualmente
Ordenar coleção Sort-Object Nome Por qualquer propriedade
Agrupar por valor Group-Object Status Agrupa por campo comum
Medir estatísticas Measure-Object Conta, soma, média, mínimo, máximo
Comparar duas listas Compare-Object $lista1 $lista2 Mostra as diferenças
Unir strings com separador Join-String -Property Nome -Separator ", " PowerShell 7+

📤 Output e Exportação

O que fazer Comando(s) Observação
Mostrar texto no terminal Write-Host "Olá mundo" Mensagem visual
Limpar o ecrã do terminal Clear-Host ou cls Limpa a consola
Apresentar como tabela Format-Table Melhora leitura
Apresentar como lista Format-List * Mostra tudo em detalhe
Gravar texto num ficheiro "texto" \| Out-File ficheiro.txt Grava logs simples
Exportar para CSV Export-Csv ficheiro.csv -NoTypeInformation Ideal após filtragem com Select-Object
Duplicar output Tee-Object -FilePath "log.txt" Guarda e continua pipeline
